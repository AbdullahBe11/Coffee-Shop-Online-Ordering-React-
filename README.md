## üìñ Project Overview
This project is a fully responsive online ordering system for a local coffee shop. It allows customers to browse a dynamic menu, view product details, and place orders. Administrators can manage the menu (Add, Update, Delete items) through a secured backend.

The system is built using the **MERN Stack** (MySQL, Express, React, Node.js) and deployed on the cloud.

---

## ‚ú® Key Features
* **Responsive Design:** Works perfectly on Mobile, Tablet, and Desktop.
* **Dynamic Menu:** Real-time fetching of coffee products from the database.
* **Admin Dashboard:** Interfaces to Create, Read, Update, and Delete (CRUD) menu items.
* **Image Uploads:** Handled via Multer and stored on the server.
* **Robust Backend:** Node.js & Express API with a Connection Pool for stability.
* **Cloud Database:** MySQL hosted on Aiven.
